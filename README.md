# LinguaLab Premium · Portugués A1

Aplicación HTML educativa para practicar portugués A1 con una ruta progresiva.

## Características

- 80 unidades progresivas.
- 320 lecciones.
- Interfaz bilingüe: español / inglés.
- Práctica oral con voz del navegador.
- Reconocimiento de voz cuando el navegador lo permite.
- Checklist automático.
- Modo repaso inteligente.
- Panel docente.
- Exportación de progreso en CSV.
- Progreso guardado con `localStorage`.

## Corrección aplicada

Se corrigieron las lecciones de diálogo para que la respuesta **B** también aparezca en portugués.  
Ejemplo corregido:

```text
A: Olá · B: Olá! Tudo bem.
```

## Cómo subir a GitHub Pages

1. Crea un repositorio nuevo en GitHub.
2. Sube estos archivos:
   - `index.html`
   - `.nojekyll`
   - `README.md`
3. Entra a **Settings > Pages**.
4. En **Source**, selecciona:
   - Branch: `main`
   - Folder: `/root`
5. Guarda y espera a que GitHub publique la página.

## Recomendación sobre micrófono

Para que el permiso del micrófono se recuerde mejor, usa la app desde GitHub Pages o cualquier sitio HTTPS.  
Si la abres como archivo local (`C:/...`), el navegador puede volver a pedir permiso.
